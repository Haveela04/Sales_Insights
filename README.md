## Overview

* End-to-end data analytics project
* Goal: analyze sales performance across regions, products, and time
* Covers data cleaning, SQL analysis, and interactive dashboarding
* Built for business decision-making and recruiter review

## Dataset

* Sales transaction dataset
* Typical fields include:

  * Order Date
  * State / Region / Market
  * Product / Category
  * Sales Quantity
  * Revenue
  * Cost / Profit
* Dataset is loaded and cleaned using Python before analysis

## Tools & Technologies

* **Python**: Data loading and cleaning
* **MySQL**: Data storage and SQL analysis
* **Power BI**: Data visualization and dashboard creation
* **GitHub**: Version control and project documentation

## Project Steps

1. Load raw dataset using Python
2. Clean data (handle nulls, data types, duplicates)
3. Export cleaned data to MySQL
4. Write SQL queries to answer business questions
5. Connect Power BI to MySQL
6. Build interactive dashboard
7. Extract insights from visuals

## SQL Analysis

* Revenue by state and region
* Profit and loss analysis
* Monthly and yearly sales trends
* Top and bottom performing products
* States running at a loss

## Power BI Dashboard

* KPIs: Total Revenue, Total Profit, Profit Margin
* Visuals include:

  * Revenue by State
  * Profit by Market
  * Monthly Sales Trend
  * Top Products
* Filters for date, region, and market
* Designed for quick business insights

## Results & Insights

* Identified high-performing and low-performing states
* Highlighted loss-making regions needing attention
* Observed monthly sales trends and seasonality
* Provided clear view of overall business performance

## How to Run This Project

* Clone the repository
* Run Python script to clean the dataset
* Import cleaned data into MySQL
* Execute SQL queries from the `.sql` file
* Open the Power BI `.pbix` file
* Refresh data and explore the dashboard
